# SpringBoot tutorial with loan service

### loan service

### docker build with jib
> jib gradle.build setup

### k8s with skaffold
> docker desktop -> enable kubernetes
> install kubectl 
> > brew install kubectl
> install skaffold cli 
> > brew install skaffold
> > https://skaffold.dev/

### build & Run
> jibDockerBuild
> skaffold dev

### k8s command
> kubectl get svc
> kubectl get pods
